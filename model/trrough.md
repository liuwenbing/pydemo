# 异常
- 广义上的错误分为错误和异常
- 错误指的是可以人为避免
- 异常是指在语法逻辑正确的逻辑下，出现的问题
- 在python中，异常是一个类，可以处理和使用

#关于自定义异常
- 只要raise异常，推荐自定义异常
- 自定义异常的时候，一般不包括一次啊内容：
    - 自定义发生异常的异常代码
    - 自定义发生异常后的提示问题
    - 自定义异常发生的行数
- 最终的目的是，一旦发生异常，方便程序员快速定位错误现场
